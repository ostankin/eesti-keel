# Материалы для изучения эстонского языка с нуля.

Эстонский язык подобен величественному, но скалистому острову. С какой стороны ни подойди -- внизу отвесные скалы грамматических исключений и сложных для запоминания правил. Многие попытки взобраться наверх заканчиваются, едва успев начаться: мотивация тает, столкнувшись с кажущимся хаосом.

Этот набор материалов -- попытка наметить относительно пологую тропинку, что ведёт если не на самую вершину, то как минимум на просторное плато, с которого открывается вид на всю красоту языка, и откуда уже не так страшно двигаться дальше.

Материалы, которые вы здесь найдете, созданы на чистом энтузиазме. Их цель -- не дать исчерпывающее академическое знание, а помочь сделать первые осмысленные шаги, минуя первоначальный частокол исключений и сконцентрировавшись на логичных и часто используемых конструкциях.

Присоединяйтесь к восхождению!
